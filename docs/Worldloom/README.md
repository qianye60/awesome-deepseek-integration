# WorldLoom

<div align="center">

</div>

WorldLoom is an AI-driven multi-world text adventure RPG. Players can create characters across different settings such as cultivation, sci-fi, and wasteland worlds, then progress through their own timelines by entering free-form actions or choosing AI-generated action options.

WorldLoom supports DeepSeek and other large language models for opening generation, story progression, action option generation, character state updates, NPC memory, and dynamic situation tracking. AI outputs are converted into structured state updates and saved persistently, allowing attributes, inventory, relationships, maps, and events to keep affecting future adventures.

## Features

- **Multi-world storytelling**: each world has its own rules, attributes, and narrative style.
- **Free-form player actions**: players can type any action or choose AI-generated options.
- **Persistent RPG state**: character attributes, inventory, event flags, NPC relationships, and memories are saved over time.
- **Dynamic situation tracking**: the AI keeps local situations updated as the timeline changes.
- **DeepSeek integration**: DeepSeek and other LLMs can power story generation, openings, and interactive gameplay.
