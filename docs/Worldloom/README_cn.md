# 织界 WorldLoom

<div align="center">
<img src="./assets/icon.png" width="128" alt="织界 WorldLoom Logo">
</div>

织界 WorldLoom 是一款 AI 驱动的多世界文字冒险 RPG。玩家可以在修仙、科幻、废土等不同世界观中创建角色，自由输入行动或选择 AI 生成的行动选项，推动属于自己的时间线。

织界支持接入 DeepSeek 等大模型，用于开局生成、剧情推进、行动选项生成、角色状态更新、NPC 记忆和动态局势呈现。游戏会将 AI 生成结果转化为结构化状态并写入存档，让角色属性、物品、关系、地图与事件在后续冒险中持续生效。

## 核心特性

- **多世界叙事**：不同世界拥有独立设定、属性体系和叙事风格。
- **自由行动输入**：玩家可以输入任意行动，也可以选择 AI 生成的行动选项。
- **长期状态存档**：角色属性、物品、事件标记、NPC 关系和记忆会持续保存。
- **动态局势追踪**：AI 会根据当前时间线生成局部局势，让世界保持变化。
- **DeepSeek 集成**：可通过 DeepSeek 等大模型驱动剧情、开局与交互体验。
